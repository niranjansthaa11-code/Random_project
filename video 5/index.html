2. The use of table in html

we use the table tag to organize the things in the html . The table without any css is like :

Employees details
Name Phone Address
Rameshor 9878787878 Ranibas,Gulmi
Shresh 985709873 Kathmandu,Nepal

Here is the code of the following above table : 

<table border="1" >
    <caption> Employees details</caption>
    <thead>
        <tr> 
            <th>Name</th>
            <th>Phone</th>
            <th>Address</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th>Rameshor</th>
            <th>9878787878</th>
            <th>Ranibas,Gulmi</th>
        </tr>
        <tr>
            <th>Shresh</th>
            <th>985709873</th>
            <th>Kathmandu,Nepal</th>
        </tr>
    </tbody>
    <tfooter>
    </tfooter>
</table>

Here the table is formed by the table tag <table> and </table>.
The caption tag above is used for telling what the table is about i.e (Employees detailes given above in the code)
The <tr> and </tr> is used for forming a new row and <th> and </th> is used for adding elements in that row simultaneously .
the colspan tag tells how much rows or coloumns of space the element would occupy in the table(i.e )

  • colspan = How many coloumn it would occupy
  • colspan = How many rows it would occupy
Here is the demo code and the result :
Demo code : 
<th colspan="2" rowspan="1"> Rameshor</th>

</ol>
For the unodered list this code was used :
<ul>

    <dt>CSS</dt>

</dl>

Customization of the bullet points / Html list

For unodered list :

        <ul type = "......">
use variable "disc" for the round default
use variable "square " for the square bullet
use variable "circle" for the circle bullet

For odered list :

        <ol type = "....">
1 is for the numerical value in type variable
i is for the small roman value in type variable
I is for the capital roman value in type variable
A is for the capital alphabetical value in type variable
a is for the small alphabetical value in type variable
